## 📂 核心项目展示 (Key Projects)

### 1. 电商大促 ROI 归因与 A/B 测试优化 🚀
- **文件**: `roi_ab_test.py`
- **核心描述**: 针对 10 万级订单数据进行清洗，利用 Python 准确定位影响 ROI 的核心因子，并设计 A/B 测试验证页面改版效果。
- **成果**: 实验组 CTR 提升 8.3%，业务 ROI 整体提升 18%。

### 2. 自动化业务对账与异常监控系统 (Python) ⚙️
- **文件**: `auto_reconciliation.py`
- **核心描述**: 针对电商多源异构数据对账痛点，编写 Pandas 自动化脚本实现一键核验。
- **成果**: 将人工 2 小时的核对工作压缩至 5 分钟，实现 100% 准确对账。

### 3. 长三角宏观经济指标 LSTM 预测模型 📈
- **文件**: `lstm_forecast.py`
- **核心描述**: 整合 10 年维度数据，利用循环神经网络进行时序预测。
- **成果**: 调优后模型 RMSE 降低 15%，有效支持宏观政策模拟。
