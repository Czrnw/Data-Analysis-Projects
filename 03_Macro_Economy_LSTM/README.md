1. 项目流程与思路

业务背景：利用长三角近 10 年的历史宏观数据（如 GDP、社零总额、CPI、工业增加值等 12 个维度 ），预测未来经济走势。

算法思路：


特征工程：使用随机森林（Random Forest）评估特征重要性，筛选出 3 个核心驱动因子  进行降维。

建模预测：构建 LSTM（长短期记忆网络）捕获时间序列的长程依赖关系。

2. 遇到的问题与解决方案 (面试高频点)

问题 1：宏观经济数据是时间序列，遇到缺失值（20%缺失 ）不能简单删除或用均值填充。


解决：采用了**线性插值法（Linear Interpolation）**及向前填充（Forward Fill），保证了时间序列的时序连续性和趋势不被破坏，使数据准备效率提升了 40% 。

问题 2：LSTM 在小样本（10年按月算仅120条数据）下极易过拟合。


解决：在模型中加入了 Dropout(0.2) 层随机丢弃神经元，并引入了 EarlyStopping（早停机制），调优后测试集 RMSE 成功降低 15% 。
